WoW addon: Warband bank button

- 暂时只有中文版

- 主要功能
1. 自动存取战团银行
2. 战团银行物品排序

- 物品设定规则
1. 优先特殊物品，手动拖进去的特殊物品，可以设定为【全存入】【集中到某人】，排序时放在最前面
2. 材料类物品，可以根据资料片分别设定，可以设定为【不存入】【全存入】【集中到某人】
   1. 有一个特殊分类，是【版本特殊物品】，比如【祥和之灵】【火绒盒】等，目前是手动硬编码的
   2. 一些NPC出售的专业物品，目前靠手动硬编码排除掉了
   3. 对于某些专业材料，如果设定成【集中到某人】，但是某个角色虽然不是那个人，但也有这个专业，那这个人身上的材料不会提交到战团银行里
4. 低优先特殊物品，其他都和优先物品类似，只是排序放在最后面，这部分还没测
